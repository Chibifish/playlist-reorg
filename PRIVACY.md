Privacy Policy — Playlist Reorg

Playlist Reorg is a personal tool used by a single user (its author) to organize videos saved in the author's own YouTube account.
Use of YouTube API Services. This tool uses YouTube API Services. By using it, the user agrees to the YouTube Terms of Service. Google's handling of data is described in the Google Privacy Policy.
What data is accessed. With the account owner's OAuth 2.0 authorization, the tool accesses the owner's own YouTube data only: the list of videos in the owner's playlists and basic video metadata (video IDs, titles, channel names). No other user's data is accessed.
How data is used, stored, and shared. Metadata is processed locally on the owner's computer for the sole purpose of sorting the owner's saved videos into playlists. It is stored only in local working files for the duration of the one-time migration, after which it can be deleted. It is never shared, sold, published, or transferred to any third party, and is not used for advertising or any other purpose.
Data deletion and revoking access. All locally stored data can be deleted by deleting the working files. API access can be revoked at any time via Google security settings, which immediately invalidates the tool's stored token.
